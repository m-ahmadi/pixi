Topology Front

Application Structure

Let’s start by listing the main dependencies used in this project:

Library Role
Pixi.JS WebGL Renderer.
TweenLite Animation and Tweening.
UIkit HTML Components.
jQuery General (Ajax, DOM, etc.)

Now let’s talk about the app.
The app is split into modules, let’s take a look at some of these modules, specially the more important ones.

Module Role
pixiWrap This module is responsible for providing high-level Canvas components to be used in the app, and is the only module that has access to Pixi.JS library.
topology Responsible for dealing with topology logic.
mediator This module is the application mediator, if two modules need to talk to each other, its the mediator’s job to handle that.
