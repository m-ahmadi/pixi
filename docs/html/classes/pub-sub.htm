PubSub

coPubSub()

Takes no arguments.
Returns an object that has the capability of registering subscribers (listeners), and publish (emit) events to them.
Any object can inhert from this object, so they can have these capabilities too.
Be sure to check util.extend(). 


var phone = util.extend( coPubSub() ),
	person = util.extend( coPerson() );


phone.on('ringing', function (e) {
	console.log(e);  // {callerId: 104, callerName: 'mohammad'}
	person.answerPhone();
});


phone.emit('ringing', {callerId: 104, callerName: 'mohammad'});
