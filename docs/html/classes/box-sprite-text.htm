BoxSpriteText

Creates a collection made out of three elements, a box, an image and a text.

Method Args Desc
changeColor( hexNum ) color: A hex number. Changes the color of the line.
changePoints( start, end ) start: An object containing x and y number properties.
end: An object containing x and y number properties. 		Changes the start and end points of the line.
If only one of the parameters is provided, then it only changes that point, and the the other one will be left untouched.
setOnmousedown( param, fn ) param: A value passed to the callback function.
fn:The callback function. 		Sets a handler to be fired whenever, the mousedown happens on the line.
If only one of the parameters is provided, then it only changes that point, and the the other one will be left untouched.
