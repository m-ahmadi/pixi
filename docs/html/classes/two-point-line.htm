TwoPointLine

Creates a line, that can only have two points:
A starting point, and an ending point.

Method Arg Desc
changeColor( hexNum ) color A hex number. Changes the color of the line.
changePoints( start, end ) startAn object containing x and y number properties.
end An object containing x and y number properties. 		Changes the start and end points of the line.
If only one of the parameters is provided, then it only changes that point, and the the other one will be left untouched.
setOnmousedown( param, fn ) paramA value passed to the callback function.
fnThe callback function. 		Sets a handler to be fired whenever, the mousedown happens on the line.
If only one of the parameters is provided, then it only changes that point, and the the other one will be left untouched. 

var config, line;

config = {
	start:      startPoint,		// Object containing an `x` and `y` property. default: {x: 0, y: 0}
	end:        endPoint,		// Object containing an `x` and `y` property. default: {x: 0, y: 0}
	lineWidth:  number,         // Number line width of the line. default: 4
	color:      hexNumber,      // Number (hex) color of the line. default: 0x000000 (black)
	alpha:      number,         // Number opacity of the line. 1: visible, 2: hidden. default: 1
};

line = newTwoPointLine(config);
