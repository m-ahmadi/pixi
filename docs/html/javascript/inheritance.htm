Inheritance

Since we do not use classes (constructor functions with their name's first letter capitalized), we do not use prototypes either.
If you want an oject to inherit from another object, use util.extend() function. 


function foo() {
	var a, b, c,
		d, e, f,
		g = 1,
		h = '.png',
		i = false,
		j = 0x4400FF;
}
