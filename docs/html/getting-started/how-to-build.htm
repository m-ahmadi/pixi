How to build the project.

First clone the repository and install the dev dependencies:


$ git clone git@repo.ravanertebat.com:cvas/topology-front.git
$ cd topology-front
$ npm install

After the installation has completed, you can build the project by one of these four ways:

  • Build by running an npm script.
  • Build by running the Webpack CLI.
  • Build by running grunt.
  • Build by running the TypeScript compiler through browserify and tsify.

Using npm scripts:


$ npm run build

Using Webpack CLI:


$ webpack js/src/main.js js/bundle.js

Using grunt (this way, grunt start watching the project and rebuild on file change/add/remove etc.):


$ grunt

TypeScript build with watch:


$ watchify -p tsify -e js/src/main.ts -o js/bundle.js -v


$ browserify -p tsify -e js/src/main.ts -o js/bundle.js

Using RequireJS:

App main entry: js/main
Excluded folder (ignore this folder): js/src
