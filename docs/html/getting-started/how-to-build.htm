How to build the project.

First clone the repository and install the dev dependencies:


$ git clone git@repo.ravanertebat.com:cvas/topology-front.git
$ cd topology-front
$ npm install

After the installation has completed, you can build the project by one these three ways:

  • Build by running an npm script.
  • Build by running the webpack CLI.
  • Build by running grunt.

Using npm scripts:


$ npm run build

Using webpack CLI:


$ webpack js/src/main.js js/bundle.js

Using grunt (this way, grunt start watching the project and rebuild on file change/add/remove etc.):


$ grunt
